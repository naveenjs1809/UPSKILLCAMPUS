# UPSKILLCAMPUS
# File Organizer System Using Java

## 📌 Project Overview
The File Organizer System is a Core Java application that automatically organizes files into categorized folders based on their file types. It helps users manage files efficiently, reduces desktop clutter, and improves productivity.

This project was developed as part of the **Core Java Internship Program** conducted by **Upskill Campus**, **The IoT Academy**, in collaboration with **UniConverge Technologies Pvt. Ltd. (UCT)**.

---

## 🎯 Objectives

- Organize files automatically.
- Categorize files based on extensions.
- Improve file management.
- Reduce manual effort.
- Learn Core Java file handling concepts.

---

## 🚀 Features

- Automatic file organization
- File type detection
- Folder creation
- File movement

- User-friendly execution
- Cross-platform compatibility

## ⚙ How to Run

1. Clone this repository.

```
git clone https://github.com/yourusername/upskillCampus.git
```

2. Open the project in Eclipse or IntelliJ IDEA.

3. Compile the Java files.

4. Run the Main.java file.

5. Select the folder to organize.

6. The application will automatically arrange files into their respective folders.
